Tools You Need

To type up your code you must have a text editor Atom, Notepad++, Nano, etc. You can find these on your web browser. To view your code you must have a web browser, I prefer Google Chrome and Mozilla Firefox. You also need a place you can host your website (I will go into this later).

Essential Files

Some essential files you need are an HTML, CSS, and an assets folder. The HTML file is where your website is created. The CSS file is where you make your webpage look decent. The assets folder is where you keep pictures, backgrounds, pdfs, etc.

Putting your website together with HTML

beginning

To start you website you must do many things. The first thing you must do is create an HTML file. The folder can be named anything you want but has to end in .html

.

The next step you must take is to make your document HTML. To do this you must type the expression <!doctype html>.

The next part you must do before you start inserting your information is to make an HTML tag <html> </html>.

The Head

The head is the information the computers look at from the tags you put in. To define the head you must put in the tags <head> and </head>.

The tags in the head give the computers different types of information to look at. These tags are called title, base, link, meta, script, and style. They all define information for the computers

The Body

The Body is the information the people who view your website look at. This contains tags that tell the computer how to present your information to the user. For example the body contains headers, paragraphs, images, links, videos, and other types of info.

The tag <p> means paragraph.

The tag <h1> creates a heading, h1 is a heading but is usually used for titles, h2 is a subheading for h1, h3 is a subheading for h2, etc. These headings range from 1 to 6 in size.

The image and video tag are quite obvious what they do. These are harder to make because you have to have the video or image in a folder connected to your website and you can't mess up the link.

Making your website look good with CSS

To make your website look different ways you must have a CSS file. To link a CSS file to your HTML file you must type the expression <link rel="stylesheet" href="home.css" type="text/css">.
